why todoc

Built for the
terminal native

01
Zero configuration

Install and go. Tasks live in ~/.todoc/tasks.json — plain JSON on your machine. No accounts, no sync, no setup wizard.

02
Beautiful Rich output

Every command renders polished tables with priority chips, ASCII progress bars, and consistent panel feedback via Rich.

03
Free-form Tags column

The Tags field is entirely yours. Store #hashtags, due dates, links, names, ticket IDs — any text. No enforced structure.

04
Bulk operations

Mark multiple tasks done at once: todoc done 1 4 7. Done tasks stay visible with strikethrough so progress is always clear.

05
Full-text search

Search description and tags instantly. todoc search "#bug" finds every match. Partial words included, case-insensitive.

06
Export to JSON & CSV

Back up with todoc export -o backup.json or use --format csv to open tasks in Excel or Google Sheets.

reference

Every command,
documented

Run todoc help for the full reference, or todoc --summary for a quick cheat sheet.

Creating & Viewing
addCreate a new task
listShow all tasks — done tasks have strikethrough
showFull detail panel for a single task
searchFull-text search across description and tags
sortDisplay list sorted by id, priority, or created
statsCompletion statistics + priority breakdown
Updating & Removing
doneMark one or more tasks complete — bulk supported
undoneReopen a completed task back to pending
editUpdate any field — only passed flags change
deleteRemove a single task with confirmation
clearRemove all completed tasks at once
resetWipe every task — complete reset, cannot be undone
Import & Export
exportSave tasks to JSON or CSV file
importLoad tasks back from a JSON backup
helpFull per-command reference with examples
Common flags
-p / --prioritycritical · high · medium · lowmedium
--tagsFree-form info column
--pendingOnly unfinished tasks
--completedOnly finished tasks
--bySort: id · priority · createdid
--descReverse the sort order
--force / -fSkip confirmation prompts
--formatExport format: json or csvjson
-o / --outputWrite export to a file
--summary / -sCompact command cheat sheet
